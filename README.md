# `bevy_terminal`

Bevy support for terminal for bevy

# Quickstart

Add with this
```toml
bevy_terminal = { git = "https://github.com/aMyTimed/bevy_terminal" }
```

Usage example in `examples/basic_game.rs`

```rust
cargo run --example basic_game
```
